Страницы пользовательского интерфейса:

1. Дерево проектов
   * Иерархия проектов
   * Название проекта (ссылка на информацию о проекте)
   * Теги проекта
   * Описание проекта (description.txt)
2. Информация о проекте
   * Название проекта
   * Идентификатор проекта
   * Папка проекта
   * Тип проекта (Gradle, Maven, Idea, NetBeans)
   * Описание проекта (description.txt)
   * Теги проекта
   * Команда по-умолчанию
   * Кнопка "Открыть в проводнике"
   * Кнопка "Открыть в Idea"
   * Кнопка "Открыть в NetBeans"
   * Кнопка "Открыть в текстовом редакторе"
3. Поиск (на всех страницах)
   * По названию
   * По тегу
   * По идентификатору
   * По описанию
   * По типу проекта
4. Типы проектов (редактируемый список)
   * Название типа (ссылка на поиск по типу проекта)
   * Приоритет
   * Маски
   * Команда по-умолчанию
5. Настройки
   * Корневая папка каталога
   * Команда по-умолчанию
   * Имя идентификационного файла проекта
   * Путь к исполняемому файлу Idea
   * Путь к исполняемому файлу NetBeans
   * Интервал автоматического сканирования
   * Http порт
6. Папки без идентификационного файла проектов (список)
   * Путь к папке
   * Тип проекта
   * Кнопка "Сгенерировать идентификационный файл"
   * Кнопка "Сгенерировать идентификаионный файл для всех"
7. Левое меню (ссылки)
   * Поиск
   * Карта тегов
   * Дерево проектов
   * Папки без идентификационного файла проектов
   * Типы проектов
   * Повторное сканирование каталога
   * Настройки
8. Карта тегов
   * Иерархия тегов
   * Название тега
   * Количество вхождений тега
